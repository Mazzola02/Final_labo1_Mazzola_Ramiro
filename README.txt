Gengar's Adventure
Gengar's Adventure es un juego de plataformas 2D inspirado en el popular personaje Pokémon Gengar. 
En este juego, asumes el papel de Gengar y te embarcas en una emocionante aventura a través de distintos niveles llenos de desafíos y enemigos.

Gengar's Adventure Gameplay

Características principales
Controla a Gengar a medida que avanza por niveles desafiantes y llenos de diversión.
Supera obstáculos, salta sobre plataformas y evita enemigos para llegar al final de cada nivel.
Enfrenta a jefes poderosos.
Experimenta una jugabilidad suave y gráficos coloridos.


Requisitos del sistema

Python 3.7 o superior.
Biblioteca Pygame instalada. Puedes instalarla usando el siguiente comando:
shell
pip install pygame



Instrucciones de instalación

Clona este repositorio en tu máquina local:
git clone https://github.com/tu_usuario/gengar-adventure.git

Navega hasta el directorio del juego:

cd gengar-adventure
Ejecuta el juego:
python main.py
